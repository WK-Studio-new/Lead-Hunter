# ⚡ LeadHunter Pro v2 — GitHub Pages

Ferramenta de geração de leads 100% no navegador, sem servidor.

## 🚀 Deploy no GitHub Pages

1. Crie um repositório (ex: `leadhunter-pro`)
2. Faça upload do `index.html`
3. **Settings → Pages → Deploy from branch → main → / (root)**
4. Acesse `https://SEU_USUARIO.github.io/leadhunter-pro`

## 🔑 Google Places API

1. Acesse [console.cloud.google.com](https://console.cloud.google.com)
2. Ative **Places API**
3. Crie uma **Chave de API**
4. Restrinja a chave para `https://SEU_USUARIO.github.io/*`

Cota gratuita: **$200/mês** (~6.000 buscas sem custo).

## ✨ Funcionalidades v2

- 57 nichos pré-cadastrados
- 27 estados + capitais do Brasil
- Seleção por estado/capital ou cidade manual
- Score automático por ausência de site, avaliações e presença digital
- Mensagens personalizadas por perfil do lead
- Export CSV direto no navegador
- API Key salva no localStorage
